# gcc-fn-spike

Spiking Google Cloud Functions with ClojureScript.
 
## Usage

Build and deploy:

```bash
./build-and-deploy.sh
```

Invoke:

```bash
http post https://europe-west1-jarppe.cloudfunctions.net/helloworld/ name=Jarppe
```
